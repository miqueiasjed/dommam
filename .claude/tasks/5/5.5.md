## Especificação

### O que fazer:

Criar `tests/Feature/Download/DownloadTest.php` com os seguintes casos de teste:

**Cenários obrigatórios:**

1. `test_download_sem_sessao_retorna_401`
   - Faz GET `/api/roteiros/{slug}/download` sem cookie de sessão
   - Assertar: `$response->assertStatus(401)`

2. `test_download_com_assinatura_inativa_retorna_403`
   - Cria sessão válida mas mock do `LastlinkService` retorna `assinaturaAtiva() = false`
   - Assertar: `$response->assertStatus(403)`

3. `test_download_com_sessao_valida_retorna_pdf`
   - Cria Roteiro publicado com `pdf_url` apontando para arquivo de teste
   - Mock do `Storage::get()` retorna bytes de um PDF mínimo válido
   - Mock do `LastlinkService` retorna `assinaturaAtiva() = true`
   - Assertar: `$response->assertStatus(200)`
   - Assertar: header `Content-Type` == `application/pdf`
   - Assertar: header `Content-Disposition` contém `attachment`

4. `test_download_registra_no_banco`
   - Repete cenário do teste 3
   - Assertar: `Download::count() == 1`
   - Assertar: download tem `email` correto e `roteiro_id` correto

5. `test_download_roteiro_nao_publicado_retorna_404`
   - Cria Roteiro com `status = 'rascunho'`
   - Request com sessão válida
   - Assertar: `$response->assertStatus(404)`

6. `test_alerta_disparado_apos_50_downloads`
   - Cria 51 registros em `downloads` para o mesmo email no período de 24h
   - Faz request de download
   - Assertar: `Log::shouldReceive('warning')` foi chamado
