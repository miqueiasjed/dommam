# Task 5.2 – MarcaDaguaService (geração de PDF com marca d'água)

## Objetivo
Instalar a biblioteca `setasign/fpdi` e implementar o `MarcaDaguaService` que recebe o PDF master em bytes e retorna um PDF personalizado com marca d'água em cada página identificando o assinante.

## Especificação

### O que fazer:

**1. Instalar dependências:**
```bash
cd backend && composer require setasign/fpdi setasign/fpdf
```

**2. Criar `MarcaDaguaService`** em `App\Services\Download\`:

```php
namespace App\Services\Download;

use setasign\Fpdi\Fpdi;

class MarcaDaguaService
{
    public function aplicar(string $pdfBytes, string $email, string $nome): string
    {
        // Salva bytes em arquivo temporário
        // Usa FPDI para iterar páginas
        // Em cada página adiciona:
        //   - Rodapé centralizado: "{$nome} · {$email} · Download em {data} · ID: {hashCurto}"
        //     opacidade 30%, cor cinza médio (RGB 128,128,128), fonte Arial 8pt
        //   - Cabeçalho: "Confidencial — distribuição proibida" alinhado à direita, mesma opacidade
        // Retorna PDF resultante como string de bytes
    }

    private function gerarHashCurto(string $email): string
    {
        return strtoupper(substr(hash('sha256', $email . now()->timestamp), 0, 8));
    }
}
```

**Detalhes de implementação:**
- Usar arquivo temporário (`tempnam(sys_get_temp_dir(), 'pdf')`) para entrada e saída
- `FPDI::setSourceFile()` com o temp file de entrada
- Loop `for ($pagina = 1; $pagina <= $totalPaginas; $pagina++)` importando cada página
- Aplicar texto com `SetTextColor(128, 128, 128)` + `SetAlpha(0.3)` se disponível (FPDF não tem alpha nativo — usar `SetFillColor` com branco transparente como alternativa ou text_color com alpha via alpha layer)
- Alternativa prática: usar cor cinza claro `RGB(180, 180, 180)` sem transparência para máxima compatibilidade
- Limpar arquivos temporários com `register_shutdown_function` ou `finally`

### Critérios de aceitação:
- [ ] `composer require setasign/fpdi setasign/fpdf` executado com sucesso
- [ ] Classe `MarcaDaguaService` criada em `App\Services\Download\`
- [ ] Método `aplicar(string $pdfBytes, string $email, string $nome): string` retorna bytes de PDF válido

- [ ] Arquivos temporários são limpos após processamento
