## Especificação

### EmailService.php — novos métodos:

Manter o método `enviarMagicLink()` existente (funciona com HTML inline).
Adicionar dois novos métodos usando Blade views:

```php
public function enviarBoasVindas(string $email, string $linkPrimeiroAcesso): void
{
    Mail::send('emails.boas-vindas', ['link' => $linkPrimeiroAcesso], function ($message) use ($email) {
        $message
            ->to($email)
            ->subject('Bem-vindo ao Danuzio History Backstage! 🎬');
    });

    Log::info('E-mail de boas-vindas enviado', ['email' => $email]);
}

public function enviarCancelamento(string $email): void
{
    Mail::send('emails.cancelamento', [], function ($message) use ($email) {
        $message
            ->to($email)
            ->subject('Sentiremos sua falta — Danuzio History Backstage');
    });

    Log::info('E-mail de cancelamento enviado', ['email' => $email]);
}
```

### Template: `emails/magic-link.blade.php`
HTML cinematográfico fundo preto (#030406), texto zinc-300, botão amber:
- Título: "Danuzio History Backstage"
- Corpo: "Clique no botão abaixo para acessar o portal. O link expira em **15 minutos**."
- Botão amber: "Acessar agora" → {{ $link }}
- Fallback link em texto
- Rodapé: "Se você não solicitou este link, ignore este e-mail."
- Variável disponível: `$link`

### Template: `emails/boas-vindas.blade.php`

### Template: `emails/cancelamento.blade.php`

- Título: "Sentiremos sua falta"
