# Task 4.5 – Rota POST /api/webhooks/lastlink + config services.lastlink

## Objetivo
Registrar a rota pública do webhook em `api.php` e garantir que a chave `services.lastlink.webhook_secret` está mapeada no config.

## Tipo
backend

## Arquivos a criar/modificar
- `backend/routes/api.php` → adicionar rota webhook
- `backend/config/services.php` → adicionar chave webhook_secret

## Contexto necessário (arquivos para ler antes)
- `backend/routes/api.php` → ver rotas existentes
- `backend/config/services.php` → ver configurações Lastlink existentes

## Skills necessárias
- laravel-arquitetura

## Especificação

### api.php — adicionar ao final do arquivo:
```php
use App\Http\Controllers\Webhook\WebhookController;

// Webhook Lastlink (rota pública — HMAC valida autenticidade)
Route::post('webhooks/lastlink', [WebhookController::class, 'receber']);
```

### config/services.php — adicionar/atualizar entrada lastlink:
```php
'lastlink' => [
    'url'            => env('LASTLINK_API_URL', 'https://lastlink.com/api/v1'),
    'api_key'        => env('LASTLINK_API_TOKEN', ''),
    'cache_ttl'      => env('LASTLINK_CACHE_TTL', 600),
    'webhook_secret' => env('LASTLINK_WEBHOOK_SECRET', ''),
],
```

### Critérios de aceitação:
- [ ] `POST /api/webhooks/lastlink` existe e é acessível (sem middleware de autenticação)
- [ ] `config('services.lastlink.webhook_secret')` retorna valor do env `LASTLINK_WEBHOOK_SECRET`
- [ ] Rota não requer autenticação (webhook é público, autenticado por HMAC)
