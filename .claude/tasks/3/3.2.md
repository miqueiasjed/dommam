## Especificação

### O que fazer:

**AdminAuthService.php:**
- Namespace: `App\Services\Admin`
- Método `tentarLogin(string $email, string $senha): array`
  - Busca AdminUser por email + ativo = true
  - Verifica senha com `Hash::check()`
  - Se TOTP não configurado: retorna `['requer_2fa' => false, 'admin' => $admin]`
  - Se TOTP configurado: gera `pending_admin_token` (signed URL ou cache com TTL 5min) e retorna `['requer_2fa' => true, 'pending_token' => $token]`
  - Lança `AuthenticationException` se credenciais inválidas ou admin inativo
- Método `verificar2FA(string $pendingToken, string $codigoTotp): AdminUser`
  - Recupera admin do cache pelo pending token
  - Valida TOTP com `OTPHP\TOTP::create($admin->totp_secret)->verify($codigoTotp)`
  - Retorna o AdminUser se válido
  - Lança `AuthenticationException` se inválido ou expirado
- Método `gerarTotpSecret(): string`
  - Usa `OTPHP\TOTP::create()->getSecret()` — retorna secret para setup inicial

**Nota:** instalar `spomky-labs/otphp` via composer se não estiver no projeto.

**LoginAdminRequest.php:**
- Namespace: `App\Http\Requests\Admin`
- Rules: email (required|email), senha (required|string|min:6)
- Messages em português

**AutenticadoComoAdmin.php:**
- Verifica cookie/sessão de admin: `session('admin_autenticado')` contém admin_id
- Carrega AdminUser pelo id e injeta em `$request->adminAutenticado`
- Atualiza `ultimo_acesso` no admin
- Retorna 401 JSON se não autenticado: `{'mensagem': 'Acesso não autorizado'}`

**bootstrap/app.php:**
- Registrar alias `auth.admin` → `AutenticadoComoAdmin::class`

### Critérios de aceitação:
- [ ] `AdminAuthService::tentarLogin()` lança exceção com credenciais erradas
- [ ] `AdminAuthService::tentarLogin()` retorna `requer_2fa: true` quando TOTP configurado
- [ ] `AdminAuthService::verificar2FA()` valida código TOTP corretamente
- [ ] Middleware retorna 401 JSON quando não há sessão de admin
- [ ] Middleware injeta `$request->adminAutenticado` quando autenticado
- [ ] Alias `auth.admin` registrado em bootstrap/app.php
