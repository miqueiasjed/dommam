# Task 3.3 – AdminAuthController + Rotas /admin/auth

## Objetivo
Criar o controller de autenticação admin com os endpoints POST /admin/login e POST /admin/2fa/verificar, e registrar as rotas em api.php.

## Tipo
config

## Arquivos a criar/modificar
- `backend/app/Http/Controllers/Admin/AdminAuthController.php` → criar
- `backend/routes/api.php` → modificar (adicionar grupo /admin)

## Contexto necessário (arquivos para ler antes)
- `backend/app/Http/Controllers/Auth/AuthController.php` → ver padrão de controller do projeto
- `backend/app/Services/Admin/AdminAuthService.php` → ver interface do serviço (Task 3.2)
- `backend/routes/api.php` → ver estrutura atual das rotas

## Skills necessárias
- laravel-arquitetura

## Especificação

### O que fazer:

**AdminAuthController.php:**
- Namespace: `App\Http\Controllers\Admin`
- Extende `BaseApiController` (ou `Controller`)
- Injeta `AdminAuthService` via construtor
- Método `login(LoginAdminRequest $request): JsonResponse`
  - Chama `$this->adminAuthService->tentarLogin($request->email, $request->senha)`
  - Se `requer_2fa = false`: inicia sessão direto (`session(['admin_autenticado' => $admin->id])`) e retorna 200 com dados do admin
  - Se `requer_2fa = true`: retorna 200 com `{'requer_2fa': true, 'pending_token': '...'}`
  - Captura exceção de auth e retorna 401 com mensagem
- Método `verificar2fa(Request $request): JsonResponse`
  - Valida: pending_token (required|string), codigo (required|string|size:6)
  - Chama `$this->adminAuthService->verificar2FA($request->pending_token, $request->codigo)`
  - Inicia sessão: `session(['admin_autenticado' => $admin->id])`
  - Atualiza `ultimo_acesso` no admin
  - Retorna 200 com dados do admin (sem sensitive fields)
  - Captura exceção e retorna 401
- Método `logout(Request $request): JsonResponse`
  - Limpa `session()->forget('admin_autenticado')`
  - Retorna 200 com mensagem de sucesso

**api.php — adicionar grupo /admin:**
```php
// Rotas de autenticação admin (públicas)
Route::prefix('admin')->group(function () {
    Route::post('login', [AdminAuthController::class, 'login']);
    Route::post('2fa/verificar', [AdminAuthController::class, 'verificar2fa']);

    // Rotas protegidas — admin autenticado
    Route::middleware('auth.admin')->group(function () {
        Route::post('logout', [AdminAuthController::class, 'logout']);
        // Plano 3 continuará adicionando rotas aqui
    });
});
```

### Critérios de aceitação:
- [ ] `POST /api/admin/login` com credenciais válidas retorna 200
- [ ] `POST /api/admin/login` com credenciais inválidas retorna 401
- [ ] `POST /api/admin/2fa/verificar` com código válido inicia sessão e retorna admin
- [ ] `POST /api/admin/logout` limpa a sessão admin e retorna 200
- [ ] `php artisan route:list` mostra as rotas admin sem erros
