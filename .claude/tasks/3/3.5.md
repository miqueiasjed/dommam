## Especificação

### O que fazer:

**RoteiroService.php:**
- Namespace: `App\Services\Conteudo`
- Injeta: `AuditLogService`, `FileUploadService`
- Método `criar(array $dados, AdminUser $admin): Roteiro`
  - Gera slug único a partir do título: `Str::slug($dados['titulo'])`
  - Se `capa_file` presente: chama `FileUploadService::uploadCapa()`
  - Se `pdf_file` presente: chama `FileUploadService::uploadPdf()`
  - Define status baseado em `publicar_em`:
    - Sem data: `status = 'rascunho'`
    - Com data futura: `status = 'agendado'`
    - Com data passada: `status = 'publicado'`, `publicado_em = now()`
  - `criado_por = $admin->id`
  - Cria Roteiro no banco
  - Chama `AuditLogService::registrar($admin, 'criar', $roteiro, null, $roteiro->toArray())`
  - Retorna Roteiro
- Método `atualizar(Roteiro $roteiro, array $dados, AdminUser $admin): Roteiro`
  - Guarda `$dadosAntes = $roteiro->toArray()`
  - Processa novos arquivos se enviados (substitui anteriores chamando `FileUploadService::remover()`)
  - Atualiza campos e recalcula status se `publicar_em` mudou
  - Chama `AuditLogService::registrar($admin, 'atualizar', $roteiro, $dadosAntes, $roteiro->fresh()->toArray())`
  - Retorna roteiro atualizado
- Método `despublicar(Roteiro $roteiro, AdminUser $admin): Roteiro`
  - Seta `status = 'rascunho'`, `publicado_em = null`
  - Registra auditoria com ação 'despublicar'
  - Retorna roteiro
- Método `agendar(Roteiro $roteiro, \Carbon\Carbon $dataPublicacao, AdminUser $admin): Roteiro`
  - Seta `status = 'agendado'`, `publicado_em = $dataPublicacao`
  - Registra auditoria com ação 'agendar'
  - Retorna roteiro
- Método `listar(array $filtros = []): \Illuminate\Pagination\LengthAwarePaginator`
  - Inicia query com `Roteiro::query()`
  - Filtro `status`: aplica `scopePorStatus()`
  - Filtro `busca`: where titulo LIKE %busca%
  - Ordena por `created_at` desc (padrão)
  - Pagina: 20 por página

**CriarRoteiroRequest.php:**
- Rules: titulo (required|string|max:255), descricao (nullable|string), conteudo (nullable|string), capa_file (nullable|image|max:5120), pdf_file (nullable|mimes:pdf|max:51200), publicar_em (nullable|date|after:now)
- Messages em português

**AtualizarRoteiroRequest.php:**
- Mesmas rules de CriarRoteiroRequest porém titulo é `sometimes|required`
- publicar_em sem `after:now` (permite datas passadas para publicação imediata)

### Critérios de aceitação:
- [ ] `RoteiroService::criar()` gera slug único sem conflito
- [ ] `RoteiroService::criar()` define status correto baseado em publicar_em
- [ ] `RoteiroService::listar()` retorna paginação com filtros funcionando
- [ ] `RoteiroService::despublicar()` muda status para 'rascunho'
- [ ] FormRequests validam e rejeitam dados inválidos com mensagens em português
