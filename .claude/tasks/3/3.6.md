## Especificação

### O que fazer:

**AdminRoteiroController.php:**
- Namespace: `App\Http\Controllers\Admin`
- Injeta `RoteiroService` via construtor
- Método `index(Request $request): JsonResponse`
  - Extrai filtros: status, busca do $request
  - Chama `$this->roteiroService->listar(['status' => ..., 'busca' => ...])`
  - Retorna paginação 200
- Método `store(CriarRoteiroRequest $request): JsonResponse`
  - Chama `$this->roteiroService->criar($request->validated(), $request->adminAutenticado)`
  - Retorna 201 com roteiro criado
- Método `show(int $id): JsonResponse`
  - Busca `Roteiro::findOrFail($id)` (incluindo soft deletes? não — apenas ativos)
  - Retorna 200 com roteiro
- Método `update(AtualizarRoteiroRequest $request, int $id): JsonResponse`
  - Busca `Roteiro::findOrFail($id)`
  - Chama `$this->roteiroService->atualizar($roteiro, $request->validated(), $request->adminAutenticado)`
  - Retorna 200 com roteiro atualizado
- Método `destroy(int $id, Request $request): JsonResponse`
  - Busca `Roteiro::findOrFail($id)`
  - Chama `softDelete` via `$roteiro->delete()`
  - Registra auditoria (acao 'deletar') via `AuditLogService`
  - Somente role 'admin' pode deletar — retorna 403 se operator
  - Retorna 200 com mensagem de sucesso
- Método `despublicar(int $id, Request $request): JsonResponse`
  - Busca roteiro, chama `$this->roteiroService->despublicar()`
  - Retorna 200 com roteiro atualizado

**api.php — adicionar dentro do grupo `auth.admin`:**
```php
Route::prefix('roteiros')->group(function () {
    Route::get('/', [AdminRoteiroController::class, 'index']);
    Route::post('/', [AdminRoteiroController::class, 'store']);
    Route::get('{id}', [AdminRoteiroController::class, 'show']);
    Route::put('{id}', [AdminRoteiroController::class, 'update']);
    Route::delete('{id}', [AdminRoteiroController::class, 'destroy']);
    Route::patch('{id}/despublicar', [AdminRoteiroController::class, 'despublicar']);
});
```

### Critérios de aceitação:
- [ ] `GET /api/admin/roteiros` sem auth retorna 401
- [ ] `GET /api/admin/roteiros` com auth admin retorna lista paginada
- [ ] `POST /api/admin/roteiros` cria roteiro e retorna 201
- [ ] `DELETE /api/admin/roteiros/{id}` por operator retorna 403
- [ ] `PATCH /api/admin/roteiros/{id}/despublicar` muda status para rascunho
- [ ] `php artisan route:list | grep admin/roteiros` mostra todas as 6 rotas
