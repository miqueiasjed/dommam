# Task 9.7 — AdminAuditLogPage

## Objetivo
Página de log de auditoria com filtros por tipo de ação e data.

## Arquivos a criar
- `frontend/src/pages/admin/AdminAuditLogPage.jsx`

## Lógica
```js
// GET /api/admin/audit-log?tipo={}&data_inicio={}&data_fim={}
// Dados: [{ id, admin_email, acao, alvo, alvo_id, criado_em }]
// Paginação simples: últimas 100 ações
```

## Visual

### Header
- "Audit Log" + "Últimas 100 ações administrativas"

### Filtros (linha)
- Select tipo: Todos | criou_roteiro | editou_roteiro | publicou_roteiro | despublicou_roteiro | deletou_roteiro | login | logout
- Date inputs: De / Até

### Lista (não tabela — cards compactos)
Cada item:
```
[ação badge] admin@email.com  •  "editou_roteiro"  •  alvo: "Título do roteiro"
                                                              2025-05-13 14:32
```
- Container: `bg-[#0d0d0f] border border-amber-500/10 rounded-xl divide-y divide-amber-500/5`
- Ação badge: cores diferentes por tipo (criar=emerald, editar=amber, deletar=red, login=zinc)
- Timestamp: `font-mono text-[10px] text-zinc-600`
- Estado vazio: "Nenhuma ação registrada"
