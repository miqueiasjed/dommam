# Task 8.5 – Componentes `RoteiroHero` e `RoteiroNav`

## Objetivo
Criar os dois componentes visuais da página de roteiro: o hero cinematográfico com a capa e o componente de navegação prev/next entre roteiros.

## Tipo
componente-dominio

## Arquivos a criar/modificar
- `frontend/src/components/roteiro/RoteiroHero.jsx` → criar
- `frontend/src/components/roteiro/RoteiroNav.jsx` → criar

## Contexto necessário (arquivos para ler antes)
- `frontend/src/index.css` → tokens CSS do projeto
- `frontend/src/components/roteiro/RoteiroCard.jsx` → padrão de estrutura de componente de roteiro

## Skills necessárias
- frontend-design-system

## Especificação

### `RoteiroHero.jsx`

**Props:** `{ capa_url, titulo, data_publicacao, temas, instagram_url }`

**Layout:**
- Container relativo: `relative w-full` com altura `h-[400px]` (ou `h-[45vh] min-h-[300px]`)
- Imagem de fundo: `object-cover w-full h-full` via `<img>` com `alt=""`
- Gradiente inferior sobre a imagem: `absolute inset-0 bg-gradient-to-t from-[#030406] via-[#030406]/60 to-transparent`
- Conteúdo posicionado na parte inferior do hero: `absolute bottom-0 left-0 right-0 p-8`
- Título: `text-white font-bold text-[28px] md:text-[36px] leading-tight`
- Data: `text-muted text-[13px] mt-1` — formatar como "13 de maio de 2026" usando `Intl.DateTimeFormat`
  ```js
  new Intl.DateTimeFormat('pt-BR', { day: 'numeric', month: 'long', year: 'numeric' }).format(new Date(data_publicacao))
  ```
- Temas (badges): usar componente `Tag` se disponível, senão `<span className="rounded-full bg-amber-900/30 px-2.5 py-0.5 text-[10px] font-medium text-amber-400">`
- Link Instagram: `flex items-center gap-1.5 text-[12px] text-zinc-400 hover:text-white transition-colors mt-3`
  - Ícone `Instagram` do lucide-react + texto "Ver no Instagram"
  - `target="_blank" rel="noopener noreferrer"`
  - Mostrar apenas se `instagram_url` fornecido

**Fallback:** se `capa_url` for null/undefined, exibir div com `bg-zinc-900` como fundo

### `RoteiroNav.jsx`

**Props:** `{ anterior, proximo }` onde cada um é `{ slug, titulo } | null`

**Layout:**
- Container: `flex items-center justify-between px-8 py-6 border-t border-zinc-800/80`
- Cada lado usa `<Link>` do react-router-dom
- Item anterior (esquerda):
  ```
  ← Roteiro anterior
  [Título do roteiro anterior]
  ```
  - Ícone `ArrowLeft` + texto "Roteiro anterior" (text-muted, text-[11px])
  - Título abaixo (text-zinc-300, text-[13px], font-medium, line-clamp-1)
- Item próximo (direita, alinhado à direita):
  ```
  Próximo roteiro →
  [Título do próximo roteiro]
  ```
  - Mesmo padrão, ícone `ArrowRight` à direita
- Hover: `hover:text-white transition-colors`
- Se `anterior` for null: lado esquerdo vazio (renderizar `<div />`)
- Se `proximo` for null: lado direito vazio

### Regras visuais aplicáveis:
- Usar tokens CSS: `bg-page`, `text-muted`, `border-default`
- Gradient: pode usar hex `#030406` pois é o mesmo valor que o token `bg-page` (consistência visual)
- Tema escuro fixo
- Nunca hardcodar outras cores

### Critérios de aceitação:
- [ ] `RoteiroHero` exibe imagem, gradiente inferior, título, data formatada e tags
- [ ] `RoteiroHero` exibe link Instagram somente quando `instagram_url` fornecido
- [ ] `RoteiroHero` exibe fallback `bg-zinc-900` quando sem capa
- [ ] `RoteiroNav` exibe setas e títulos para anterior e próximo
- [ ] `RoteiroNav` oculta o lado correspondente quando `anterior` ou `proximo` for null
