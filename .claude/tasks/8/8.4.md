# Task 8.4 – `WelcomeTourPage`

## Objetivo
Criar a tela de boas-vindas com tour de 3 slides exibida apenas no primeiro acesso. Após concluir ou pular, o usuário é redirecionado para `/biblioteca` e a flag no localStorage impede reexibição.

## Tipo
pagina

## Arquivos a criar/modificar
- `frontend/src/pages/WelcomeTourPage.jsx` → criar
- `frontend/src/App.jsx` → modificar (adicionar rota `/boas-vindas`)

## Contexto necessário (arquivos para ler antes)
- `frontend/src/App.jsx` → entender estrutura de rotas para adicionar `/boas-vindas`
- `frontend/src/index.css` → tokens CSS do projeto
- `frontend/src/components/ui/Button.jsx` → props disponíveis

## Skills necessárias
- frontend-design-system

## Especificação

### O que fazer:

**1. Criar `frontend/src/pages/WelcomeTourPage.jsx`**

Conteúdo dos 3 slides:
```js
const slides = [
  {
    icone: BookOpen,  // lucide-react
    titulo: 'Bem-vindo ao Backstage',
    descricao: 'Acesse os roteiros completos do @danuzio_history com 1 hora de antecedência, antes de qualquer pessoa.',
  },
  {
    icone: FileText,
    titulo: 'Roteiros em PDF',
    descricao: 'Baixe os roteiros em PDF com marca d\'água personalizada. Salve para assistir offline quando quiser.',
  },
  {
    icone: Library,
    titulo: 'Sua biblioteca',
    descricao: 'Todos os episódios organizados por tema e data. Use os filtros para encontrar o que procura.',
  },
];
```

**Estado:**
- `slideAtual`: número (0, 1, 2)

**Navegação:**
- Indicadores de ponto (dots) para cada slide
- Botão "Próximo" avança slide
- No último slide, botão muda para "Começar" → salva flag + redireciona
- Link "Pular tour" no canto superior direito → salva flag + redireciona

**Flag localStorage:**
```js
const TOUR_KEY = 'danuzio_tour_concluido';
// Ao concluir/pular:
localStorage.setItem(TOUR_KEY, '1');
navigate('/biblioteca', { replace: true });
```

**Verificação no mount (useEffect):**
```js
// Se flag já existe, redirecionar para /biblioteca imediatamente
if (localStorage.getItem(TOUR_KEY)) {
  navigate('/biblioteca', { replace: true });
}
```

**Layout:**
- Tela cheia: `h-screen bg-page flex flex-col items-center justify-center`
- Container do slide: `max-w-md w-full px-8 text-center`
- Ícone: `amber-500`, tamanho 48, centralizado
- Título: `text-white font-semibold text-[20px] mt-6 mb-3`
- Descrição: `text-muted text-[14px] leading-relaxed`
- Dots: bolinhas pequenas (`w-2 h-2 rounded-full`), ativa = `bg-amber-500`, inativa = `bg-zinc-700`
- Botões: usar componente `Button` (primario/ghost conforme contexto)
- "Pular tour": botão ghost pequeno, posicionado `absolute top-6 right-6`

**2. Adicionar rota em `App.jsx`:**
```jsx
import WelcomeTourPage from './pages/WelcomeTourPage';
// ...dentro de <Routes>:
<Route path="/boas-vindas" element={<WelcomeTourPage />} />
```
Não precisa de ProtectedRoute (o redirect do callback já garante autenticação).

### Regras visuais aplicáveis:
- Usar tokens CSS: `bg-page`, `text-muted`
- Nunca hardcodar cores hex
- Acento dourado: `amber-500` nos ícones e dot ativo
- Tema escuro fixo

### Critérios de aceitação:
- [ ] Slide 1 exibido ao entrar em `/boas-vindas` pela primeira vez
- [ ] Navegar pelos 3 slides com botão "Próximo"
- [ ] Botão "Começar" no slide 3 salva flag e redireciona para `/biblioteca`
- [ ] "Pular tour" salva flag e redireciona para `/biblioteca`
- [ ] Segunda visita a `/boas-vindas` redireciona imediatamente para `/biblioteca`
- [ ] Rota `/boas-vindas` presente no App.jsx
