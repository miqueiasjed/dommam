## Especificação

### SearchBar.jsx

**Props:** `{ busca, onBuscaChange }`

```jsx
// Campo de busca com ícone de lupa
<div className="relative">
  <Search size={14} className="absolute left-3 top-1/2 -translate-y-1/2 text-zinc-500" />
  <input
    type="text"
    value={busca}
    onChange={e => onBuscaChange(e.target.value)}
    placeholder="Buscar roteiros..."
    className="w-full rounded-lg border border-zinc-800 bg-surface pl-8 pr-3 py-2 text-[12px] text-zinc-300 placeholder:text-zinc-600 focus:border-zinc-700 focus:outline-none"
  />

```
- O debounce real é feito no hook `useRoteiros` — este componente só chama `onBuscaChange` imediatamente

### SortSelector.jsx

**Props:** `{ ordenacao, onOrdenacaoChange }`

Opções:
```js
const opcoes = [
  { valor: 'mais_recentes', rotulo: 'Mais recentes' },
  { valor: 'mais_antigos', rotulo: 'Mais antigos' },
  { valor: 'alfabetico_asc', rotulo: 'A → Z' },
  { valor: 'alfabetico_desc', rotulo: 'Z → A' },

```

```jsx
<select
  value={ordenacao}
  onChange={e => onOrdenacaoChange(e.target.value)}

  {opcoes.map(o => <option key={o.valor} value={o.valor}>{o.rotulo}</option>)}
```

### FilterBar.jsx

**Props:** `{ filtros, onFiltrosChange, roteiros }`
- `filtros: { mes: '', temas: [], tipo: '' }`
- `onFiltrosChange(novosFiltros)` — callback que substitui o objeto inteiro de filtros

**Derivar opções dos roteiros disponíveis:**
```js
// Meses únicos dos roteiros (de publicado_em)
const meses = [...new Set(roteiros.map(r => r.publicado_em?.slice(0, 7)))].sort().reverse();
// Temas únicos
const temas = [...new Set(roteiros.flatMap(r => r.tema ? [r.tema] : []))].sort();
// Tipos fixos
const tipos = ['Bastidor', 'Evento histórico', 'Personagem'];
```

**Layout:** flex row, gap-2, flex-wrap
- Dropdown de mês: `<select>` (igual ao SortSelector, com opção "Todos os meses")
- Tags de tema: botões clicáveis (múltipla seleção)
  - Selecionado: `bg-amber-600 text-white border-amber-600`
  - Normal: `border border-zinc-700 text-zinc-400 hover:border-zinc-600`
  - Classe base: `rounded-full px-3 py-1 text-[11px] font-medium transition-colors`
- Dropdown de tipo: `<select>` com opção "Todos os tipos"
- Botão "Limpar filtros": visível apenas se algum filtro está ativo
  - Usar `Button variante="ghost" tamanho="sm"` com ícone `X` do lucide
  - Ao clicar: `onFiltrosChange({ mes: '', temas: [], tipo: '' })`

**Detectar filtro ativo:**
```js
const filtroAtivo = filtros.mes || filtros.temas.length > 0 || filtros.tipo;
```

**Toggle de tema:**
```js
function toggleTema(tema) {
  const novos = filtros.temas.includes(tema)
    ? filtros.temas.filter(t => t !== tema)
    : [...filtros.temas, tema];
  onFiltrosChange({ ...filtros, temas: novos });
}
```

### Regras visuais:
