# Task 7.6 – Componente CarrosselTematico (opcional)

## Objetivo
Criar o `CarrosselTematico` — scroll horizontal de cards agrupados por categoria/tema, com setas de navegação. Componente opcional mas recomendado pelo PRD para enriquecer a biblioteca.

## Tipo
componente-dominio

## Arquivos a criar/modificar
- `frontend/src/components/roteiro/CarrosselTematico.jsx` → criar

## Contexto necessário (arquivos para ler antes)
- `frontend/src/components/roteiro/RoteiroCard.jsx` → usar como item do carrossel
- `frontend/src/index.css` → tokens de cor

## Skills necessárias
- frontend-design-system

## Especificação

### O que fazer:

Criar `frontend/src/components/roteiro/CarrosselTematico.jsx`:

**Props:**
```js
CarrosselTematico({ tema, roteiros })
// tema: string (nome do tema/categoria)
// roteiros: array de roteiros filtrados por esse tema
```

**Estrutura:**
```jsx
<section>
  {/* Título do carrossel */}
  <h2 className="mb-3 text-[14px] font-semibold text-white">{tema}</h2>

  {/* Container de scroll */}
  <div className="relative group/carrossel">
    {/* Botão prev */}
    {/* Scroll container */}
    <div
      ref={scrollRef}
      className="flex gap-3 overflow-x-auto scroll-smooth pb-2 scrollbar-hide"
    >
      {roteiros.map(r => (
        <div key={r.slug} className="w-[140px] sm:w-[160px] flex-shrink-0">
          <RoteiroCard roteiro={r} />
        </div>
      ))}
    </div>
    {/* Botão next */}
  </div>
</section>
```

**Botões de navegação:**
- Aparecem no hover do container (`group-hover/carrossel:opacity-100 opacity-0`)
- Esquerda: `ChevronLeft`, direita: `ChevronRight` do lucide-react
- Estilo: `absolute top-1/2 -translate-y-1/2 z-10 rounded-full bg-black/70 p-1.5 text-white hover:bg-black/90`
- Prev: `left-1`, Next: `right-1`
- Scroll: `scrollRef.current.scrollBy({ left: ±320, behavior: 'smooth' })`

**Esconder scrollbar nativo:**
```css
/* Adicionar em index.css */
.scrollbar-hide::-webkit-scrollbar { display: none; }
.scrollbar-hide { -ms-overflow-style: none; scrollbar-width: none; }
```

**Nota:** Ao usar no `BibliotecaPage`, agrupar roteiros por tema antes de renderizar os carrosseis. Exemplo:
```js
const temas = [...new Set(roteiros.map(r => r.tema).filter(Boolean))];
// Para cada tema, renderizar <CarrosselTematico tema={tema} roteiros={filtradosPorTema} />
```
Esta integração fica a critério do desenvolvedor que executar o Plano 8 ou como ajuste de BibliotecaPage.

### Regras visuais:
- Cards dentro do carrossel têm largura fixa (`w-[140px]`)
- Setas só aparecem no hover do carrossel inteiro
- Nunca hardcode de cor, exceto overlays com `black/70`
