## Especificação

### MagicLinkService (`backend/app/Services/Auth/MagicLinkService.php`):

**Responsabilidade:** gerar token único, salvar em `sessions`, disparar envio de e-mail.

```php
namespace App\Services\Auth;

use App\Models\Sessao;
use App\Services\Email\EmailService;
use Illuminate\Support\Facades\Log;
use Carbon\Carbon;

class MagicLinkService
{
    public function __construct(
        private readonly EmailService $emailService,
    ) {}

    public function gerar(string $email): void
    {
        // Revogar tokens anteriores não usados do mesmo e-mail
        Sessao::where('email', $email)->whereNull('usado_em')->delete();

        $token = bin2hex(random_bytes(32)); // 64 chars hex
        $tokenHash = hash('sha256', $token);

        Sessao::create([
            'email'     => strtolower(trim($email)),
            'token_hash' => $tokenHash,
            'expira_em' => Carbon::now()->addMinutes(15),
        ]);

        $this->emailService->enviarMagicLink($email, $token);

        Log::info('Magic link gerado', ['email' => $email]);
    }
}
```

**Importante:** o `token` (raw) vai no e-mail para o usuário clicar. O `token_hash` (SHA-256) fica no banco. Nunca armazenar o token raw.

### SessaoService (`backend/app/Services/Auth/SessaoService.php`):

**Responsabilidade:** verificar token na chegada, criar sessão de cookie, buscar sessão ativa de um request, revogar sessão.

```php
namespace App\Services\Auth;

use App\Models\Sessao;
use Carbon\Carbon;
use Illuminate\Http\Request;
use Illuminate\Support\Facades\Log;

class SessaoService
{
    private string $nomeCookie = 'backstage_sessao';
    private int $duracaoSessaoDias = 30;

    public function verificar(string $token): ?Sessao
    {
        $tokenHash = hash('sha256', $token);

        $sessao = Sessao::ativa()->where('token_hash', $tokenHash)->first();

        if (!$sessao) {
            return null; // token inválido ou expirado
        }

        // Marcar como usado (invalida o token para evitar reuso)
        $sessao->update(['usado_em' => Carbon::now()]);

        Log::info('Magic link verificado com sucesso', ['email' => $sessao->email]);

        return $sessao;
    }

    public function criarCookieSessao(string $email): \Symfony\Component\HttpFoundation\Cookie
    {
        // Payload simples — em produção, considerar JWT ou token criptografado
        $payload = base64_encode(json_encode([
            'email'      => $email,
            'criado_em'  => Carbon::now()->toIso8601String(),
            'expira_em'  => Carbon::now()->addDays($this->duracaoSessaoDias)->toIso8601String(),
        ]));

        return cookie(
            name: $this->nomeCookie,
            value: $payload,
            minutes: $this->duracaoSessaoDias * 24 * 60,
            path: '/',
            secure: app()->environment('production'),
            httpOnly: true,
            sameSite: 'Lax',
        );
    }

    public function sessaoAtiva(Request $request): ?object
    {
        $cookieValue = $request->cookie($this->nomeCookie);

        if (!$cookieValue) {
            return null;
        }

        try {
            $dados = json_decode(base64_decode($cookieValue));

            if (!$dados || !isset($dados->email, $dados->expira_em)) {
                return null;
            }

            if (Carbon::parse($dados->expira_em)->isPast()) {
                return null; // cookie expirado
            }

            return $dados;
        } catch (\Exception) {
            return null;
        }
    }

    public function revogar(Request $request): \Symfony\Component\HttpFoundation\Cookie
    {
        Log::info('Sessão revogada');
        return cookie()->forget($this->nomeCookie);
    }
}
```

### Regras visuais aplicáveis:
- Token raw nunca fica no banco — apenas o hash SHA-256
- Cookie `httpOnly: true` e `secure: true` em produção
- `sessaoAtiva()` retorna `null` se cookie ausente, inválido ou expirado

### Critérios de aceitação:
- [ ] `MagicLinkService::gerar()` invalida tokens anteriores do mesmo e-mail antes de criar novo
- [ ] Token raw é gerado com `random_bytes(32)` (64 hex chars); banco recebe hash SHA-256
- [ ] `SessaoService::verificar()` retorna `null` para token inválido ou já usado
- [ ] `SessaoService::verificar()` marca `usado_em` ao validar com sucesso
- [ ] `SessaoService::criarCookieSessao()` cria cookie httpOnly com duração de 30 dias
- [ ] `SessaoService::sessaoAtiva()` retorna `null` se cookie ausente, inválido ou expirado
- [ ] `SessaoService::revogar()` retorna cookie de remoção
